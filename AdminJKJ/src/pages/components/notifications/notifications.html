消息通知、提示配置参数 
		
使用方法：$.jkj.cleverNotify.error('这是消息内容',null,'self'); 来调用信息提示

		
其中：.error() 错误信息; .success() 成功信息; .info() 一般信息; .warning() 提示信息;

		
如果提示在遮罩上层，将null改为{z_index:1099}，修改z_index的值来提高它的层级

		
page的参数: self当前页面；parent父页面；top顶级页面；opener打开的页面

		
.remove(id) 移除指定提示信息

		
.removeAll() 移除所有提示信息

		   
			
			<script>
				// 默认配置信息
				defaults: {
					// 消息类型。['danger' | 'info' | 'success' | 'warning']
					type: 'info',
					// 是否显示进度条
					showProgressbar: false,
					/**
					 * 消息放置的页面。['self' | 'parent' | 'top']
					 * self： 当前页面
					 * parent：父页面
					 * top：顶级页面
					 */
					page: 'self',
					/**
					 * 显示位置
					 * from：['top' | 'bottom']
					 * align：['left' | 'center' | 'right']
					 */
					placement: {
						from: 'top',
						align: 'right'
					},
					// 消息靠边缘的间隔，居中显示时x轴无效
					offset: {
						x: 15,
						y: 5
					},
					// 消息间上下间隔像素
					spacing: 2,
					// Z轴的值
					z_index: 1031,
					// 实例化方法被调用时回调该方法
					onShow: null,
					// 消息显示到用户界面时回调该方法
					onShowed: null,
					// 消息关闭回调该方法
					onClose: null,

					onClosed: null,
					// 消息模板。{0} = type；{1} = title；{2} = message；{3} = url；{4} = target
					template: '<div data-notify="container" class="col-xs-11 col-sm-3 alert alert-{0} alert-dismissible" role="alert">' +
					'<button type="button" aria-hidden="true" class="close" data-notify="dismiss">×</button>' +

					'<a href="{3}" target="{4}" data-notify="url"></a>' +
					'</div>',

老版本notify 通知 
		
使用方法：$.jkj.notify.showError('message',null,showWindow); 来调用信息提示

		
其中：.showError() 错误信息;	 .showSuccess() 成功信息;	 .showInfo() 一般信息;	.showWarning() 提示信息;

		
如果提示在遮罩上层，将null改为{z_index:1099}，修改z_index的值来提高它的层级

		
showWindow的参数: self当前页面；parent父页面；top顶级页面；opener打开的页面
